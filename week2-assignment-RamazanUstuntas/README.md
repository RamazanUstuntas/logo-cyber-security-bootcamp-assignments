## Getting Started

### Prerequisites

Check the node package manager, install it.
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
2. Go to the directory
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run
   ```sh
   npm start
   ```
